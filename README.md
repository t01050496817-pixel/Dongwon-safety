# 동원건설산업 아침 조회 실시간 다국어 자막 시스템 (Morning Assembly Live Subtitles)

이 프로젝트는 아침 조회나 회의 시 마이크로 입력된 한국어 음성을 실시간으로 인식하여 7개국어(중국어, 영어, 베트남어, 러시아어, 우즈벡어, 캄보디아어, 미얀마어)로 번역해주는 웹 애플리케이션입니다.

## 주요 기능
- **실시간 음성 인식**: Web Speech API를 사용하여 한국어 음성을 텍스트로 변환합니다.
- **실시간 번역**: Google Cloud Translation API를 사용하여 7개 언어로 동시 번역합니다.
- **영화 자막 스타일**: 가독성 높은 다크 모드 UI와 큰 폰트를 제공합니다.

## 실행 방법
이 프로젝트는 별도의 설치 과정 없이 브라우저에서 바로 실행할 수 있습니다.

1. **Chrome 브라우저**에서 엽니다. (다른 브라우저는 Web Speech API 지원이 제한될 수 있습니다.)
2. 마이크 권한을 허용합니다.
3. **'자막 시작'** 버튼을 클릭합니다.
